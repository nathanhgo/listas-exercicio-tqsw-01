# Smoke Testing 

A *Smoke testing Technique* valida as funcionalidades mais básicas da aplicação, preferivelmente após cada alteração no código fonte.

## Vantagens

- Valida que as principais funcionalidades do seu sistema estão funcionando
- Rápidos para determinar se a nova versão está estável ou não

## Desvantagens

- Não tem uma cobertura completa do sistema por estar focado apenas nas funcionalidades mais básicas oferecidas pelo sistema.

## Exemplo

Em um aplicativo de mensagens, por exemplo, conjunto enxuto de casos de teste poderiam verificar se a funcionalidade principal de enviar mensagens está funcionando como o esperado. Esse tipo de teste não se preocuparia, por exemplo, se a funcionalidade de "Alterar foto de perfil" está operante ou não.
